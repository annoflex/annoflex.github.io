Support

Bugs and features requests can be submitted via the GitHub issues page. If you report something then please try to provide as many information as you can, especially details like JRE and OS version, stacktraces of exceptions, minimized Java classes to reproduce an issue and so on. In general, the more information is available, the faster an issue can be solved. The GitHub issue page is loacted here:
