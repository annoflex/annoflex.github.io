Support

Bug reports and feature requests can be submitted via the GitHub issues page. If you report something then please try to provide as many information as you can, especially details like JRE and OS version, stacktrace of exceptions, minimized Java class to reproduce an issue and so on. In general, the more information is available, the faster an issue can be solved.
