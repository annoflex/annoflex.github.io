Source Code

The source code of the latest version of AnnoFlex is available on GitHub at the following location:

Latest source: https://github.com/annoflex/annoflex/tree/master/source/org/annoflex

The source code of older releases can be found in the "source" folder of the corresponding download package. A list of all releases can be found here:

Older releases: https://github.com/annoflex/annoflex/releases

Please note: In contrast to many other open source projects AnnoFlex does not provide the version for each single change in the code base. Instead, all changes between two releases are provided by one single commit to the repository. This is mainly because GitHub is used as a publishing platform and not as a source code repository.
