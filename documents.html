Documents

There are three documents with important information about AnnoFlex. The first one is the readme. It describes the most important things you need to know about AnnoFlex. You should read it before you use AnnoFlex the first time. The second one is the manual. It is the main source for details about the syntax of regular expressions and scanner settings as well as the API of the code generator and the overall infrastructure of AnnoFlex. The third document is the changelog. It contains a list of changes for all versions of AnnoFlex.
