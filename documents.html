Documents

The reference documentation of AnnoFlex is the manual. All aspects of AnnoFlex are described there in detail. Additionally, there is the readme file. It describes the most important things you need to know about AnnoFlex. You should read it before you use AnnoFlex the first time. The changelog file contains a list of all changes of all versions of AnnoFlex. The following links take you to these documents:

Manual: https://github.com/annoflex/annoflex/blob/master/manual.pdf

Readme: https://github.com/annoflex/annoflex/blob/master/readme.txt

Changelog: https://github.com/annoflex/annoflex/blob/master/changelog.txt
